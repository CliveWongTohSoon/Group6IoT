1.0 Introduction
Our project consists of 4 parts: 

1.1 main.py
This python script is installed on the ESP8266 to interface with the colour sensor. The address on the writeto() and readfrom() is inferred from the data sheet provided by the colour sensor. 


1. This website must be run through FLASK framework.
2. Set it up by first installing FLASK through pip3 install Flask
3. Run it by go to the directory containing server.py, calling in terminal "FLASK_APP=server.py flask run"
